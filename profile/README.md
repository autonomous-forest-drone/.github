# Autonomous Forest Drone

> PPO-based autonomous navigation system for forest environment monitoring and bark beetle detection

## 📖 About

This project is the result of our Bachelor's Thesis in Computer Science and Engineering at **Kristianstad University** in collaboration with **BeetleSense** (Christo Van Zyl). We developed an autonomous drone system capable of navigating forest environments using **Proximal Policy Optimization (PPO)** reinforcement learning for real-time obstacle avoidance.

The system combines computer vision, reinforcement learning, and autonomous flight control to enable safe navigation through dense forest areas for ecological monitoring applications, specifically targeting bark beetle detection.
